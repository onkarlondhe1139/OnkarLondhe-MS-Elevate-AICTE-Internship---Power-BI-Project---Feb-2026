# MakeMyTrip Travel Analytics Dashboard (Power BI)

## 📊 Project Overview

The **MakeMyTrip Travel Analytics Dashboard** is a data visualization project developed using **Microsoft Power BI** to analyze travel booking data and extract actionable insights.

This dashboard helps understand **booking trends, revenue performance, popular destinations, airline usage, and traveler demographics** through interactive visualizations.

The goal of this project is to demonstrate how **data analytics and business intelligence tools** can transform raw travel data into meaningful insights for decision-making.

---

## 🎯 Objectives

* Analyze **travel booking trends over time**
* Identify **top destinations and airlines**
* Understand **customer travel behavior**
* Monitor **revenue performance**
* Provide **interactive visual insights** using Power BI

---

## 📂 Dataset

A synthetic travel booking dataset was generated for this project.

**Dataset Fields**

* Booking_ID
* Booking_Date
* Destination
* State
* Airline
* Traveler_Type
* Booking_Channel
* Booking_Value
* Customer_Rating

The dataset simulates real-world travel booking data used for analytics.

---

## 📈 Dashboard Insights

The dashboard provides the following business insights:

* **Total Bookings** – Overall booking count
* **Total Revenue** – Revenue generated from travel bookings
* **Average Booking Value** – Average transaction value
* **Customer Satisfaction Rating** – Average customer feedback score
* **Booking Trends** – Monthly travel demand analysis
* **Top Destinations** – Most popular travel locations
* **Traveler Demographics** – Distribution of travel types
* **Top Airlines** – Most used airline services
* **Geographic Analysis** – Booking distribution across India
